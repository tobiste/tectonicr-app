# tectonicr-app
Graphical user interface to tectonicr

## Run in browser:
https://tobiste.shinyapps.io/tectonicr-app/
(no need to install R and tectonicr)

## Run in RStudio
without installing:
```
shiny::runGitHub("tectonicr-app", "tobiste")
```

or download and unzip the project. Then run
```
shiny::runApp()
```
